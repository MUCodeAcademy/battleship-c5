To join a game

To join a game you must receive a game code from a player who has created a game and paste it into the join game input field.

To play battleship

The goal of battleship is to sink your opponent's fleet of five ships before they sink yours.
Rules for placing ships: Start by placing each ship in any horizontal or vertical position, using the horizontal or vertical buttons. You cannot place your ships diagonally. You will not be able to change the position and/or location of the ship once it's placed.
