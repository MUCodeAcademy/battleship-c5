Create Game

After clicking on create game you will be given a game code. Share your code with a friend so that they can join!

Join Game

To join a game you must receive a game code from a player who has already created a game. The game code is located in the first player's URL endpoint (ex: /game/12345678 --> 12345678 = game code). Copy the game code and paste it into the input field to join game.
