Create A Game

After clicking on create game you will be given a game code. Share your code with a friend so that they can join!

To Join A Game

To join a game you must receive a game code from a player who has created a game, the game code is located in the URL. Example /game/12345678 with 12345678 being your game code. Copy the game code and paste it into the input field to join game.
